## Working Product Name

Use `Concept Ladder` as a placeholder name until the team chooses a final brand.

## Core Idea

People learning complex topics often get blocked by unfamiliar terms inside explanations. They search those terms, find more unfamiliar terms, open more tabs, and lose the original thread.

Concept Ladder turns an explanation into an interactive learning path. A user enters a topic or pastes dense text, receives a clear explanation, and can click highlighted terms to drill down into contextual explanations. Each drill-down becomes part of a concept graph so the learner can go deeper without getting lost.
